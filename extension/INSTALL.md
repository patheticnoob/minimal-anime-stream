# Installation Guide

## Method 1: Load Unpacked (Development)

1. **Download the Extension**
   - Download or clone this repository
   - Locate the `extension` folder

2. **Open Chrome Extensions**
   - Open Google Chrome
   - Navigate to `chrome://extensions/`
   - Or click Menu (⋮) → More Tools → Extensions

3. **Enable Developer Mode**
   - Toggle "Developer mode" switch in the top-right corner

4. **Load the Extension**
   - Click "Load unpacked" button
   - Navigate to and select the `extension` folder
   - Click "Select Folder"

5. **Verify Installation**
   - You should see "GojoStream Controller Support" in your extensions list
   - The extension icon should appear in your toolbar

6. **Connect Your Controller**
   - Plug in or connect your gamepad via Bluetooth
   - Visit GojoStream
   - You should see a notification when the controller is detected
